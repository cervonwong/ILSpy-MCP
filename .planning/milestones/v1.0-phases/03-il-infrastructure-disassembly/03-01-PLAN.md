**File 1: `Infrastructure/Decompiler/ILSpyDisassemblyService.cs`**

Create the ReflectionDisassembler adapter implementing `IDisassemblyService`. Key implementation details:

1. Constructor takes `ILogger<ILSpyDisassemblyService>` and creates `DecompilerSettings` with same settings as `ILSpyDecompilerService` (`ThrowOnAssemblyResolveErrors = false`, `ShowXmlDocumentation = false`).

2. `DisassembleTypeAsync` implementation (per D-01 and D-02):
   - Wrap in `Task.Run(() => { ... }, cancellationToken)` like existing service
   - Create `CSharpDecompiler` from `assemblyPath.Value` with `_settings`
   - Resolve type via `decompiler.TypeSystem.MainModule.GetTypeDefinition(new FullTypeName(typeName.FullName))` -- throw `TypeNotFoundException` if null
   - Get `metadataFile` from `decompiler.TypeSystem.MainModule.MetadataFile`
   - Create `StringWriter` + `PlainTextOutput(writer)` + `ReflectionDisassembler(output, cancellationToken)`
   - Set `ShowMetadataTokens = showTokens`, `DetectControlStructure = true`
   - **Summary header (D-01):** Write comment lines: `// Type: {type.FullName}`, `// Assembly: {assemblyPath.FileName}`, `// Methods: {type.Methods.Count}`, then blank line
   - **Headers-only iteration (D-02 -- CRITICAL, do NOT call DisassembleType which includes bodies):**
     - Iterate `type.Fields` -> call `disassembler.DisassembleFieldHeader(metadataFile, (FieldDefinitionHandle)field.MetadataToken)` + `output.WriteLine()`
     - Iterate `type.Methods` -> call `disassembler.DisassembleMethodHeader(metadataFile, (MethodDefinitionHandle)method.MetadataToken)` + `output.WriteLine()`
     - Iterate `type.Properties` -> call `disassembler.DisassembleProperty(metadataFile, (PropertyDefinitionHandle)prop.MetadataToken)` + `output.WriteLine()`
     - Iterate `type.Events` -> call `disassembler.DisassembleEvent(metadataFile, (EventDefinitionHandle)evt.MetadataToken)` + `output.WriteLine()`
   - Return `writer.ToString()`
   - Catch `FileNotFoundException` -> throw `AssemblyLoadException` (same pattern as `ILSpyDecompilerService`)

3. `DisassembleMethodAsync` implementation (per D-03 and D-07):
   - Same `Task.Run` + `CSharpDecompiler` + type resolution pattern
   - Find methods matching `methodName`: `type.Methods.Where(m => m.Name == methodName).ToList()`
   - If no methods found: throw `MethodNotFoundException(methodName, typeName.FullName)`
   - If multiple overloads found: throw `MethodNotFoundException` with message listing available overloads (per D-07, matching `decompile_method` behavior)
   - Create `ReflectionDisassembler` with `ShowMetadataTokens = showTokens`, `ShowRawRVAOffsetAndBytes = showBytes`, `DetectControlStructure = true`
   - Call `disassembler.DisassembleMethod(metadataFile, (MethodDefinitionHandle)method.MetadataToken)` for the single matching method
   - Return `writer.ToString()`
